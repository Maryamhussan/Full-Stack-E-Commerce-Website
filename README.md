# E-STORE: Modern Minimalist E-Commerce Platform

A portfolio-ready, full-stack e-commerce application built with Next.js 15, TypeScript, and Tailwind CSS. Designed with a "Neutral Luxury" editorial aesthetic.

## Features
- **Immersive Brand Landing Page**: Cinematic hero section with Framer Motion animations.
- **Dynamic Catalog**: Fully searchable and filterable product grid with 12 curated items.
- **Persistent Shopping Cart**: React Context-based cart persistence using LocalStorage.
- **Favorites System**: User-specific product saving.
- **User Authentication**: Mock login/register system.
- **Seamless Checkout**: Professional multi-step checkout workflow with order summary.
- **Responsive Architecture**: Fully fluid layout optimized for all screen sizes.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4 (Neutral Luxury Theme)
- **State**: React Context API
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Getting Started
1. Install dependencies: `npm install`
2. Start development: `npm run dev`
3. View at: `http://localhost:3001`
